The original shaded map [2] was enhanced with albedo details [1] of the correct contrast. Suitable for low poly models.

Geometric albedo value (0.0435) was used, using spherical one (0.0174) would make the map much darker [3].

The commercial license allows it to be used in open source projects such as Celestia.



Info

Simple cylindrical projection, planetocentric coordinate system, center longitude 0°.

Gamma corrected, albedo corrected.



Sources
